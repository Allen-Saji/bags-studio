# Bags API Mapping

## Core Bags Surfaces

### Token / Creator Context
Use Bags endpoints that expose token launch creators and associated metadata to anchor coin ownership and creator context.

### Analytics / Health
Use Bags analytics endpoints for:
- token lifetime fees
- claim stats
- claim events
- pool / state details

These help derive momentum health signals and post-launch activity snapshots.

### Fee Share
Use fee share and partner configuration endpoints as a future extension for routing supporter incentives and creator reward programs.

### Trade / Swap
Potential stretch integration for campaign-linked onboarding and interaction flows.

## Product Use
Bags data should support:
- coin dashboard bootstrapping
- creator identity and context
- activity and momentum calculations
- campaign milestone logic
- future value-routing mechanisms
