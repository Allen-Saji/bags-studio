# MVP Scope

## Must Have
- select Bags coin
- creator dashboard
- supporter score computation
- supporter leaderboard
- create one campaign type
- supporter-facing campaign page
- reward eligibility output

## Nice to Have
- multiple campaign types
- automated reward flows
- richer analytics
- revival mode suggestions
- fee-share integrations

## Non-Goals for V1
- fully generalized marketing CRM
- full token launch flow
- complex AI orchestration
- broad multi-chain support
